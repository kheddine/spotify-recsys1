HOW IT WORKS (The Magic):
================================================================================

mockData.js:
  - Has 15 songs (Blinding Lights, Levitating, etc.)
  - When user enters mood, it matches keywords
  - Returns relevant recommendations
  - All happens in the browser (no server!)

App-GitHubPages.jsx:
  - Uses mockData instead of API calls
  - Let users create playlists
  - Get recommendations
  - Add/remove tracks
  - Everything works offline

GitHub Pages:
  - Hosts your React app for free
  - Super fast (uses CDN)
  - Auto-deploys when you run "npm run deploy"
  - URL: https://kheddine.github.io/spotify-recsys1

================================================================================
WHAT HAPPENS WHEN YOU RUN "npm run deploy":
================================================================================

1. React builds optimized code
2. Creates build/ folder with all files
3. "gh-pages" package pushes to GitHub
4. GitHub creates "gh-pages" branch
5. GitHub Pages serves those files
6. Your site goes live at the URL
7. Takes 1-2 minutes
