# Quick GitHub Pages Setup (3 Steps)

## You have: `https://kheddine.github.io/spotify-recsys1`

---

## STEP 1: Update `frontend/package.json`

Add this ONE line after `"version": "1.0.0"`:

```json
{
  "name": "spotify-recsys",
  "version": "1.0.0",
  "homepage": "https://kheddine.github.io/spotify-recsys1",
  ...
}
```

Then add these scripts (replace the existing scripts section):

```json
"scripts": {
  "start": "react-scripts start",
  "build": "react-scripts build",
  "deploy": "npm run build && gh-pages -d build",
  "predeploy": "npm run build",
  "test": "react-scripts test",
  "eject": "react-scripts eject"
}
```

---

## STEP 2: Install GitHub Pages Package

```bash
cd frontend
npm install --save-dev gh-pages
```

---

## STEP 3: Deploy!

```bash
cd frontend
npm run deploy
```

Done! âœ…

Your site will be live at: **`https://kheddine.github.io/spotify-recsys1`**

Wait 1-2 minutes and refresh!

---

## Files You Need to Add to `frontend/src/`

### 1. Copy `mockData.js` to `frontend/src/`

This file has all the track data and recommendation logic.

### 2. Copy `App-GitHubPages.jsx` to `frontend/src/App.jsx`

Replace the existing App.jsx with this version that uses mock data.

### 3. Keep your other components:
- `components/ChatInterface.jsx`
- `components/PlaylistViewer.jsx` 
- `components/RecommendationCard.jsx`
- `App.css`

## Troubleshooting

**"gh-pages not found"**
```bash
npm install --save-dev gh-pages
npm run deploy
```

**Site not updating**
- Wait 2-3 minutes
- Hard refresh browser (Ctrl+Shift+R)

**404 errors**
Make sure homepage is set correctly in package.json
